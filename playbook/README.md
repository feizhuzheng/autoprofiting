# AutoProfiting Trading Playbook

An autonomous AI agent is learning to trade US stocks with $100,000 in paper money. This repository contains its **evolving trading playbook** — the accumulated wisdom from every trade, every mistake, and every reflection.

**Live site:** [autoprofiting.com](https://autoprofiting.com)
**Playbook page:** [autoprofiting.com/strategy](https://autoprofiting.com/strategy)
**Trading journal:** [autoprofiting.com/journal](https://autoprofiting.com/journal/)

## What is this?

Every 30 minutes, the AI agent:
1. Analyzes the market (news, prices, trends)
2. Reflects on past decisions (what worked, what didn't)
3. Makes trading decisions (buy, sell, hold)
4. Updates this playbook with new insights

The playbook is the AI's most valuable asset. It grows with every session and directly informs future trading decisions.

## Playbook Structure

**`strategy.json`** contains:

| Section | Description |
|---------|-------------|
| `principles` | Core trading beliefs learned from experience |
| `lessons` | Specific insights with context and action items |
| `mistakes` | Honest record of what went wrong and how to fix it |
| `what_works` | Strategies and patterns that have proven successful |
| `open_questions` | Things the AI is still figuring out |

## Why open source?

- **Transparency**: See exactly how an AI thinks about markets
- **Learning**: The lessons here apply to human traders too
- **Accountability**: Public playbook = no hiding from mistakes
